# The TriFold Timetraveller 🕰️

一款富有创意的Unity游戏项目，融合了梦幻场景、时间旅行和冒险元素。

## 📖 项目简介

《The TriFold Timetraveller》是一个独立游戏项目，由多个梦境关卡和精心设计的场景组成。玩家将穿越不同的时间和空间，体验独特的游戏故事。

## 🎮 游戏特性

- 🎨 **丰富的美术资源** - 多套精心设计的角色和场景美术
- 🎵 **动态音乐系统** - 多首背景音乐和音效
- 🗂️ **多个梦境章节** - Dream1、Dream2等不同场景

- 🎭 **场景设计** - GameKeyObject、家具系统等交互要素

## 📂 项目结构

```
Assets/
├── Art/                 # 美术资源
│   └── sprites/        # 精灵图表
├── Characters/         # 角色资源
├── Dream1/             # 第一梦章节
├── Dream2/             # 第二梦章节
├── Music/              # 背景音乐和音效
├── Prefab/             # 游戏预制体
├── Scenes/             # 游戏场景
├── Scripts/            # 核心脚本代码
└── GameKeyObject/      # 游戏关键对象
    ├── Furniture/      # 家具系统
    └── Room/           # 房间组件
```

## 🚀 快速开始

### 系统要求
- Unity 2020 LTS 或更高版本
- 建议使用 Windows 或 macOS

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/YOUR_USERNAME/TheTriFoldTimetraveller.git
   cd TheTriFoldTimetraveller
   ```

2. **在 Unity 中打开项目**
   - 打开 Unity Hub
   - 选择 "Add Project"
   - 选择项目文件夹
   - 等待项目导入完成

3. **运行游戏**
   - 在 Scenes 文件夹中找到起始场景
   - 按 Play 开始游戏

## 🎯 主要场景和脚本

- **IntroSceneController.cs** - 介绍场景控制器
- **Dream2Inventory.cs** - 梦境2的物品系统
- 其他场景特定脚本位于各梦境文件夹中

## 🎨 美术资源

- 角色精灵图表存储在 `Characters/` 文件夹
- UI美术资源位于 `Art/sprites/`
- 游戏对象预制体在 `GameKeyObject/` 中
